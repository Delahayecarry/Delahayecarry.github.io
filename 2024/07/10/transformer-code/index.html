Transformer类

定义编码器、解码器和输出层结构

以下是结构的定义：

super主要用于调用父类(超类)的一个方法。self.encoder = Encoder()是构造Encoder的大框架，transformer分为几个大框架：为编码器、解码器和输出层。这里 self.encoder，self.decoder和self.projection是我们构造的大框架（可以理解为菜篮子，里面是空的，我们要往里面装东西），而 Encoder()，Decoder()和nn.Linear是我们要装进去的东西，这三个装进去的东西我们将会在后面单独写一个类。

根据数据流通方向编写

以下是根据数据流通的方向做描述（面向对象编程）：

enc_inputs和 dec_inputs是指在transformer模型对于Encoder和decoder两个模块的输入。

先看encoder端的输入：词向量经过位置编码后输入到encoder中，经过encoder中存在的多头注意力模块，输出到前馈神经网络(Feed Forward)，再输出到decoder模块，这两个关键模块后面会详细将到，因为只是进行框架的构建，详细先不展开。

从右往左看代码，enc_inputs输入到encoder模块中得到 enc_outputs, enc_self_attns分别是encoder输出和encoder的自注意力。enc_self_attns是Q与K相乘后得到矩阵经过softmax的矩阵，即每个单词和其他单词包括自己的相关性。

再看decoder端输入：同样是词向量经过位置编码输入到decoder中，经过带masked的多头注意力模块，与encoder的输入混合再输入到多头注意力模块，然后经过前馈神经网络输出到线性连接层，经过softmax层后输出。

dec_inputs, enc_outputs, enc_self_attns输入decoder后输出 dec_outputs, dec_self_attns, dec_enc_attns ，现在只需知道这些参数怎么来的。

dec_logits 是通过 self.projection 层对 self.decoder 的输出 dec_outputs 进行线性变换得到的。这里的 self.projection 是一个 nn.Linear 层，它的作用是将解码器的输出（dec_outputs）映射到目标词汇表大小的维度上。
