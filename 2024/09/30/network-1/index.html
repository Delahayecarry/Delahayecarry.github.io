关于近期鼓捣服务器的记录

一些行业黑话

灵车

灵车指的是非常容易跑路的服务商。

挂机宝

挂机宝是一种小型服务器或者小虚拟机，用于长时间执行某项任务。如挂机游戏，自动化脚本运行等。

DD

指换系统，用非平台的方式更换服务器的操作系统。（有些VPS提供商禁止DD）

杜甫、毒妇

是指独立服务器。

ONEMAN

指一个人经营的服务器提供商。非常容易跑

小鸡

VPS，虚拟服务器。

母鸡

独立服务器。因为vps都是拿独立服务器虚拟出来的（其实就是独服开了一堆虚拟机），所以小vps叫小鸡，生出小vps来的独服叫母鸡。

超兽

超售，指售出的份额超过服务器承载能力，如假设一个独立服务器一般虚拟10台vps是正常情况，而一些商家为了利益最大化，会虚拟20台乃至以上的vps来出售。

传家宝

指一些性价比极高且稀少的服务器，往往伴随着极高的溢价。

落地鸡

网络不好，但是解锁啥的比较好，做落地。

中转机

一般指线路优秀，或者国内沿海城市的中转服务器。

NAT鸡

指的是共享ipv4的鸡。用法就是多一层端口转发。虚拟机的NAT模式，运营商的公网地址(NAT上网)。

MJJ

小鸡是vps，母鸡是独立服务器，而购买他们的人就是买鸡鸡的人，简称MJJ。

石头盘、钻石盘

指一些IO极低的服务器的硬盘，钻石盘（030M/S）是比石头盘（30100M/S）更低IO的硬盘。

CF

美国公司Cloudflare缩写，CDN服务商，提供很多免费服务，堪称业界良心，因其提供强大且免费的功能而备受MJJ推崇。但是现在基本上是万人骑。

乌龟壳

Oracle Cloud（甲骨文云）提供永久免费VPS，配置高，速度快，线路好，需要信用卡注册，失败率极高，注册成功也容易被封号，无数MJJ趋之若鹜，因其成功注册的配置页面与乌龟壳相似而得名，同时其严格的风控导致很难注册，也印证了其龟壳硬的特性。

送中

VPS的IP被谷歌送中, 意思就是VPS IP被谷歌标记成中国大陆的IP了。

IP被谷歌定位成中国的影响

  • Youtube Premium 不能用 (YouTube Premium 和 YouTube Music Premium 不支持中国)
  • Google play不能下载软件
  • Youtube 不支持后台播放

IP被谷歌定位中国 可能的原因

浏览器或APP开了GPS定位(比如手机/电脑)权限, 使用VPS的IP访问谷歌服务时, Google可能把GPS定位和IP关联起来, 导致IP被标记为中国.

判断 IP是否被谷歌定位中国 的方法

直接打开 www.youtube.com/red

IP被谷歌定位中国的解决方案

  • 关闭浏览器或手机APP的定位(GPS)权限
    • 电脑移除定位权限: Chrome – 设置 – 隐私设置和安全性, 网站设置 – 位置信息, 移除谷歌的相关站点.
    • 手机关闭APP的定位权限: 应用的权限管理, 定位服務/位置信息 的权限.
  • 去Google的帮助中心, 报告 IP 问题 一般提交后, 可能1周以上才会有效果(当然也不一定被采纳), 可以间隔1周提交一次.
  • 电脑端的谷歌浏览器, 通过Location Guard插件, 强制标记GPS定位. 安装后, Fixed Location菜单, 标记需要模拟GPS定位的位置, 根据自己VPS IP, 在地图上找到对应的国家/城市. Options菜单, Default level: 改成 Use fixed location. 经常使用Google搜索/油管等服务, 长期给谷歌推送定位.

原生IP

IP地址，在IP数据库里面的标注信息中，有两个重要属性：地理位置和用途。地理位置不用多说，大家都懂。而IP用途分类就点花样。VPS业界常将可以访问本地服务，播放当地地域限定内容的IP称为原生IP。 可是，这种原生性并不是IP地址的固有属性，而是IP数据库对IP的标注，导致提供服务网站查询这些IP数据库，给具有特定地理位置和业务用途属性的IP地址的准入许可。

什么是Vmess？

VMess 是一个基于 TCP 的加密传输协议，所有数据使用 TCP 传输，是由 V2Ray 原创并使用于 V2Ray 的加密传输协议，它分为入站和出站两部分，其作用是帮助客户端跟服务器之间建立通信。在 V2Ray 上客户端与服务器的通信主要是通过 VMess 协议通信。

Vmess 是一种基于 WebSocket 协议的加密传输协议，它允许用户通过加密的方式在互联网上安全地传输数据。Vmess 协议通常用于虚拟专用网络（VPN）服务、代理服务等，以保护用户的隐私和数据安全。

Vmess 的主要特点包括：

  1. 加密传输：使用 TLS/SSL 加密，确保数据在传输过程中不被窃听或篡改。
  2. 兼容性：Vmess 协议与多种客户端和服务器软件兼容，易于部署。
  3. 灵活性：支持多种传输方式，如 TCP、UDP、mKCP、WebSocket 等，可以根据需要选择合适的传输方式。

什么是Ws?

WebSocket 是一种网络通信协议，提供了在单个TCP连接上进行全双工通信的能力。它被设计用来满足web页面和服务器之间需要进行实时数据交换的场景，比如在线游戏、实时通讯应用、股票行情更新等。

WebSocket 的主要特点包括：

  1. 全双工通信：WebSocket 允许服务器主动向客户端发送消息，而不需要客户端先发送请求。这与HTTP协议的请求-响应模式不同。
  2. 持久连接：一旦WebSocket连接建立，它会保持开放状态，直到客户端或服务器决定关闭它。这减少了建立和关闭连接的开销。
  3. 低延迟：由于连接是持久的，WebSocket 可以减少通信的延迟。
  4. 数据格式灵活：WebSocket 可以发送文本和二进制数据，这使得它适用于各种类型的应用。
  5. 跨域通信：WebSocket 支持跨域通信，允许客户端与不同域的服务器建立连接。

WebSocket 的使用过程通常包括以下几个步骤：

  1. 握手：客户端通过发送一个特殊的HTTP请求来发起WebSocket连接。如果服务器支持WebSocket，它会响应一个握手请求，一旦握手成功，就建立了WebSocket连接。
  2. 数据传输：连接建立后，客户端和服务器就可以通过这个连接发送和接收数据了。
  3. 关闭连接：当通信结束时，任何一方都可以发送一个关闭连接的信号，然后关闭WebSocket连接。

WebSocket 协议的典型用途包括：

  • 实时通知和消息
  • 多用户游戏
  • 协作工具
  • 股票行情实时更新
  • 任何需要服务器和客户端之间实时交互的场景

Vless+Reality

vless是什么？

VLESS 和 VMess 都是 SSR 和 V2Ray 协议的一种变体，它们的主要区别在于数据加密方式、传输方式、握手协议等方面。

但是，vless采用了更加安全的AEAD加密方式，vmess使用的是AES-CFB等对称加密方式。

VLESS 使用了更加安全的 TLS1.3 协议进行握手，而 VMess 则使用 TLS1.2。TLS1.3 协议在安全性和效率方面都有所提高。

VLESS 采用了更加高效的 QUIC 协议作为传输方式，而 VMess 则采用的是 TCP 或者 WebSocket。QUIC 协议是一种基于 UDP 协议的传输方式，相比 TCP 协议可以更好地适应不稳定的网络环境，提高传输效率。

Reality是什么？

Reality 协议是一个相对较新的网络协议，它旨在提供一种更加安全、高效和灵活的方式来进行数据传输。Reality 协议通常用于构建虚拟专用网络（VPN）、代理服务等，以帮助用户绕过网络限制、保护隐私和提高数据传输的安全性。

REALITY 在流行代理协议中是最强的，强于 TLS。（不代表一定安全）

CN2?

CN2（China Telecom Next Generation Carrier Network）是中国第二大电信运营商——中国电信的下一代承载网。CN2 是中国电信为满足日益增长的互联网和数据通信需求而建立的一张高速、大容量、高质量的网络，它旨在提供更稳定、更快速的互联网服务。

CN2 的主要特点包括：

  1. 高速传输：CN2 网络采用了先进的技术，如 MPLS（多协议标签交换），以提供高速的数据传输。
  2. 低延迟：CN2 网络优化了路由策略，减少了数据包的传输时间，从而降低了延迟。
  3. 高可靠性：CN2 网络设计了多重备份和冗余机制，以确保网络的稳定性和可靠性。
  4. 覆盖广泛：CN2 网络不仅覆盖中国大陆，还通过国际海底光缆和卫星通信系统，连接到全球多个国家和地区。
  5. 安全性：CN2 提供了包括加密传输在内的多种安全措施，以保护用户数据的安全性。
  6. QoS（Quality of Service）：CN2 能够根据不同的业务需求提供差异化的服务质量保证。

CN2 网络主要服务于对网络质量要求较高的政企客户、数据中心以及高端个人用户。它通常被用于承载重要的企业应用、数据中心互联、云服务、视频会议、在线游戏、高清视频流等业务。

各个线路可以想象成中国的各级道路，而CN2GIA就是网络上的高速公路。
